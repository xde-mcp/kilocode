# Setting Up Mistral for Free Autocomplete

This guide walks you through setting up Mistral's Codestral model for free autocomplete in Kilo Code. Mistral offers a free tier that's perfect for getting started with AI-powered code completions.

## Step 1: Navigate to Mistral Console

Go to [console.mistral.ai](https://console.mistral.ai) and sign up or log in to your account.

![Mistral Console](./autocomplete-mistral-setup/CleanShot%202026-01-12%20at%2016.41.43.png)

## Step 2: Access API Keys

Navigate to the API Keys section in the Mistral console.

![API Keys Section](./autocomplete-mistral-setup/CleanShot%202026-01-12%20at%2016.42.30.png)

## Step 3: Create a New API Key

Click to create a new API key for use with Kilo Code.

![Create API Key](./autocomplete-mistral-setup/CleanShot%202026-01-12%20at%2016.43.06.png)

## Step 4: Copy Your API Key

Copy the generated API key. Make sure to save it somewhere safe as you won't be able to see it again.

![Copy API Key](./autocomplete-mistral-setup/CleanShot%202026-01-12%20at%2016.43.40.png)

## Step 5: Open Kilo Code Settings

In VS Code, open Kilo Code and navigate to the settings.

![Kilo Code Settings](./autocomplete-mistral-setup/CleanShot%202026-01-12%20at%2016.44.32.png)

## Step 6: Find Autocomplete Settings

Locate the autocomplete configuration section.

![Autocomplete Settings](./autocomplete-mistral-setup/CleanShot%202026-01-12%20at%2016.45.03.png)

## Step 7: Select Mistral Provider

Choose Mistral as your autocomplete provider.

![Select Mistral](./autocomplete-mistral-setup/CleanShot%202026-01-12%20at%2016.45.36.png)

## Step 8: Enter Your API Key

Paste your Mistral API key into the configuration field.

![Enter API Key](./autocomplete-mistral-setup/CleanShot%202026-01-12%20at%2016.46.02.png)

## Step 9: Verify Configuration

Confirm that your API key is properly configured.

![Verify Configuration](./autocomplete-mistral-setup/CleanShot%202026-01-12%20at%2016.46.20.png)

## Step 10: Enable Autocomplete

Enable the autocomplete feature to start using it.

![Enable Autocomplete](./autocomplete-mistral-setup/CleanShot%202026-01-12%20at%2016.47.00.png)

## Step 11: Start Coding!

You're all set! Autocomplete suggestions will now appear as you type.

![Ready to Use](./autocomplete-mistral-setup/CleanShot%202026-01-12%20at%2016.47.31.png)
