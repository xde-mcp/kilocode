# Setting Up Mistral for Free Autocomplete

This guide walks you through setting up Mistral's Codestral model for free autocomplete in Kilo Code. Mistral offers a free tier that's perfect for getting started with AI-powered code completions.

## Step 1: Open Kilo Code Settings

In VS Code, open the Kilo Code panel and click the **Settings** icon (gear) in the top-right corner.

![Open Kilo Code Settings](./autocomplete-mistral-setup/CleanShot%202026-01-12%20at%2016.41.43.png)

## Step 2: Add a New Configuration Profile

Navigate to **Settings â†’ Providers** and click **Add Profile** to create a new configuration profile for Mistral.

![Add Configuration Profile](./autocomplete-mistral-setup/CleanShot%202026-01-12%20at%2016.42.30.png)

## Step 3: Name Your Profile

In the "New Configuration Profile" dialog, enter a name like "Mistral profile" and click **Create Profile**.

![Create Mistral Profile](./autocomplete-mistral-setup/CleanShot%202026-01-12%20at%2016.43.06.png)

## Step 4: Select Mistral as Provider

In the **API Provider** dropdown, search for and select **Mistral**.

![Select Mistral Provider](./autocomplete-mistral-setup/CleanShot%202026-01-12%20at%2016.43.40.png)

## Step 5: Get Your API Key

You'll see a warning that you need a valid API key. Click **Get Mistral / Codestral API Key** to open the Mistral console.

![Get API Key Button](./autocomplete-mistral-setup/CleanShot%202026-01-12%20at%2016.44.32.png)

## Step 6: Navigate to Codestral in Mistral AI Studio

In the Mistral AI Studio sidebar, click **Codestral** under the Code section.

![Select Codestral](./autocomplete-mistral-setup/CleanShot%202026-01-12%20at%2016.45.03.png)

## Step 7: Generate API Key

On the Codestral page, find the **API Key** section and click **Generate API Key**.

![Generate API Key](./autocomplete-mistral-setup/CleanShot%202026-01-12%20at%2016.45.36.png)

## Step 8: Confirm Key Generation

Click the **Generate API Key** button to create your new Codestral API key.

![Confirm Generate](./autocomplete-mistral-setup/CleanShot%202026-01-12%20at%2016.46.02.png)

## Step 9: Copy Your API Key

Once generated, click the **copy** button next to your API key to copy it to your clipboard.

![Copy API Key](./autocomplete-mistral-setup/CleanShot%202026-01-12%20at%2016.46.20.png)

## Step 10: Paste API Key in Kilo Code

Return to Kilo Code settings and paste your API key into the **Mistral API Key** field.

![Paste API Key](./autocomplete-mistral-setup/CleanShot%202026-01-12%20at%2016.47.00.png)

## Step 11: Save Your Settings

Click **Save** to apply your Mistral configuration. You're now ready to use free autocomplete!

![Save Settings](./autocomplete-mistral-setup/CleanShot%202026-01-12%20at%2016.47.31.png)
